# Archive Docs

该目录用于存放历史版本资料、对比报告与快照清单。  
这些文件用于追溯与参考，不参与系统运行。

> 状态：ARCHIVED（历史资料，部分内容可能与当前实现不一致）

## 目录说明

- `reports/`：阶段性报告、项目展示与方案文档
- 其他历史清单目录（如 `history/`、`inventory/`）按需启用，默认不作为运行依赖

## 主要文档

- `reports/路由对比报告.md`
- `reports/路由修复最终报告.md`
- `reports/虚空系统项目展示PPT.md`
- `reports/框架与系统开发策划案.md`
- `reports/虚空学习系统_PPT提取.txt`

## 使用建议

- 对外介绍项目时，优先引用根目录 `README.md`
- 需要说明项目背景时，可补充引用 `docs/project-concept.md`
- 仅在需要追溯历史决策时再查阅本目录内容
